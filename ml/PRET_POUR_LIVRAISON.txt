## 🚀 DÉMARRAGE RAPIDE

```bash
# 1. Installer dépendances
pip install -r ml/requirements.txt

# 2. Entraîner modèle (optionnel, modèle pré-entraîné existe)
python -m ml.models.train_models

# 3. Lancer microservice
python -m ml.api.app

# 4. Tester (dans un autre terminal)
python ml/tests/test_api.py
```

L'API est disponible sur **`http://localhost:5000`**

## 📞 INTÉGRATION AVEC LARAVEL

Le backend Laravel appelle le microservice sur port 5000 :

```php
// Exemple appel depuis Laravel
$response = Http::post('http://ml-service:5000/classifier', [
    'categorie' => 'médical',
    'description' => $description,
    'audio_base64' => $audio,
    'nb_signalements_similaires' => 2,
    'historique' => ['total' => 5, 'valides' => 4]
]);
```

Le microservice retourne un JSON complet avec tous les détails d'analyse ✅
